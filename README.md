# Support Ticketing System

## 📌 Project Description
A full-stack ticket raising system where users can create, manage, and track support tickets.

## 🛠 Technologies Used
- Backend: Java (Spring Boot)
- Frontend: React (Vite)
- Database: H2 / MySQL
- Tools: Git, GitHub

## ⚙️ Features
- Create tickets
- View tickets
- Update ticket status
- Delete tickets

## 🚀 How to Run

### Backend
cd backend  
mvn spring-boot:run  

Runs on: http://localhost:8080  

### Frontend
cd frontend  
npm install  
npm run dev  

Runs on: http://localhost:5173  

## 📂 Project Structure
- backend/ → Spring Boot API  
- frontend/ → React UI
